TheCircle Daily Horoscope widget for all star signs

== Description ==
The [Daily Horoscope widget from TheCircle](http://www.thecircle.com) offers the day's latest tips, opportunities and risks. The astrologers from TheCircle calculate these daily horoscopes for you using their wealth of experience and knowledge.

Choose your daily updated horoscope via your zodiac sign (Aries, Taurus, Gemini, Cancer, Leo, Virgo, Libra, Scorpio, Sagittarius, Capricorn, Aquarius, Pisces) to find out what the stars hold for you on this day.

This daily horoscope allows you to prepare for each new day!

== Installation ==
= Automated Installation =
In the WordPress admin area under "> Install Plugins",  search for "TheCircle Daily Horoscope" and install.

= Manual Installation =
Download "thecircle-daily-horoscope.zip" from "Plugins> Install> Upload" button in the WordPress admin area.  Unpack and upload the folder "thecircle-daily-horoscope.zip" to "/wp-content/plugins/"

= Activate the plugin = 
Activate the plugin under "Plugins" in the WordPress admin area

= Add this widget to the sidebar =
After activating the plugin, the widget will appear in the WordPress admin area under "-> Widgets Design". Drag the widget to the position in your sidebar that you wish the daily chart to appear.
